# Audio Files for Focus Mode

These audio files are used for the Focus Mode feature.

## Source
Audio files sourced from the moodist project.

## Structure
- `nature/` - Natural ambient sounds (river, waves, wind, etc.)
- `rain/` - Rain and thunder sounds
- `animals/` - Animal sounds (birds, crickets, etc.)
- `urban/` - City and traffic sounds
- `places/` - Location-specific ambience (cafe, airport, etc.)
- `transport/` - Vehicle sounds (train, airplane, etc.)
- `things/` - Object sounds (keyboard, clock, etc.)
- `noise/` - White, pink, and brown noise
- `binaural/` - Binaural beats (if applicable)

## Usage
In development, these files are served from `/sounds/` locally.
In production, they are served from the GitHub repository.
